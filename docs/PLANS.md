# Plans

Roadmap and execution plan tracking.

## How plans work

1. Before starting non-trivial work, create a plan in `exec-plans/active/`
2. Use `/verify plan "..."` to get external validation from Codex
3. Execute the plan
4. On completion, move to `exec-plans/completed/` with results

See [exec-plans/README.md](exec-plans/README.md) for the plan template.
